xrp address converter (proposed)

option 1








The above "xrp address" was generated by decoding the r address, appending the tag (as a 32-bit integer), and performing a Bech32 encoding (with HRP = 'xrp'). This tool has not been tested.

option 2








The tag value is offset by 1234567890 (decimal) prior to encoding, so that a tag of 0 doesn't create a visual pattern (qqq...) in the encoded address. This is just a rough proof-of-concept and likely contains bugs.
